# The German Planck Layout

The goal was to have all letters including umlauts and ß available on the base
layer without moving away too much from the default Planck base qwerty layout.

The Lower layer is a numpad replacement and contains keys useful for navigating
the Linux console or terminal multiplexers like screen and tmux.

The Raise layer contains numbers and symbols. The arrangement of the symbols is
symmetrical and aimed at programmers.

The F-Keys are on the Fun layer. That is it's sole purpose...
